Directive

Requirement

How to do it

  1. Define a directive
  2. Define template for directive
  3. Define controller for directive
  4. Using directive as a HTML tag
1. Create directive with name "searchField" 

    app.directive("searchField",function(){
     return {
            templateUrl : "search-field.html",
            controller : function ($scope){
                 $scope.getValue = function(){
                    alert($scope.modelName)
                }
            }
        }  
    })
2. Create search-field.html template           

<div>
    <input ng-model="modelName" />
    <button ng-click="getValue()">Submit</button>
</div>
3.Use directive <search-field> 

<search-field></search-field>
