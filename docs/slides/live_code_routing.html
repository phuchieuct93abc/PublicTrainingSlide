Routing

Requirement

  • Main
  • Show orders
  • Add new order

How to do it

  1. Download and include angular-route
  2. Add ngRoute as dependency
  3. Define routing strategy by $routeProvider
  4. Define template for each view
  5. Use ng-view to put the content provided by the routing.
1. Download and include the AngularJS Route module 

<script src="angular-route.js"></script>
2. Add the ngRoute as a dependency in the application module 

var app = angular.module("myApp", ["ngRoute"]);
3. Use the $routeProvider to configure different routes in your application 

app.config(function($routeProvider) {
  $routeProvider
  .when("/", {
    templateUrl : "main.html"
  })
  .when("/showOrders", {
    templateUrl : "show-orders.html"
  })  
   .when("/addOrder", {
    templateUrl : "add-order.html"
  }) 
});
4. Use ng-view to put the content provided by the routing. 

<div ng-view>Views are here</div>
