Two-way data binding

In controller, $scope is the glue that connects view and model together 

    function myController($scope){
        $scope.modelName="Value 1";
        $scope.getValue = function(){
            alert($scope.modelName);
        };                        
    }
    app.controller('myController',myController)
1. From view to model by using ng-model="modelName" or ng-click="getValue()" 

    <input ng-model="modelName">
    <button ng-click="getValue()">Get value</button>
2. From model to view by using {{modelName}} 

    <div>{{modelName}}</div>
