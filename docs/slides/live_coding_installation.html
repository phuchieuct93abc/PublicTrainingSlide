Installation

  1. Download and include AngularJS
  2. Declare ng-app with our own module
  3. Declare ng-controller with our own controller
1. Download and include AngularJS in the HTML page 

    <script src="angular.js"></script>
2.Declare module by ng-app 

    index.html
    <html ng-app="myApp">


    app.js
    var app = angular.module("myApp",[])
3. Declare controller with ng-controller 

    index.html                     
    <body ng-controller="myController">
    </body>


    app.js
    app.controller("myController",function($scope){
        //Your code here    
    })
