Two-way data binding

Requirement

 {{demoInput}}
  • {{item.name}} at {{item.time | date:"HH:mm:ss"}}

How to do it

  1. Define models and function in controller
  2. Get value from view by ng-model and ng-click
  3. Show model to view by ng-repeat and {{modelName}}
In controller, $scope is the glue that connects view and model together 

    app.controller('myController',function($scope){
        $scope.list = []
        $scope.add = function () {
            var newItem = {
                name: $scope.name,
                time: new Date()
            }
            $scope.list.push(newItem)

        }                     
    })
1. From view to model by using ng-model="modelName" or ng-click="getValue()" 

<input ng-model="name" />
{{name}}	
<button ng-click="add()">Add</button>
2. From model to view by using {{modelName}} 

    <li ng-repeat="item in list">
    {{item.name}} at {{item.time | date:"HH:mm:ss"}}
    </li>
