Routing

1. Include the AngularJS Route module 

<script src="angular-route.js"></script>
2. Add the ngRoute as a dependency in the application module 

var app = angular.module("myApp", ["ngRoute"]);
3. Use the $routeProvider to configure different routes in your application 

app.config(function($routeProvider) {
  $routeProvider
  .when("/", {
    templateUrl : "main.html"
  })
  .when("/red", {
    templateUrl : "red.html"
  })  
   .when("/green", {
    templateUrl : "green.html"
  }) 
});
4. Use ng-view to put the content provided by the routing. 

<div ng-view>Views are here</div>

    Redirect by
<a href="#red">Red</a>
<a href="#green">Green</a>
<a href="#blue">Blue</a>
