DOM Manipulation

Visibility manipulation by using ng-if or ng-show 

    <div ng-show="isShow">Content here</div> 


    In controller:
    $scope.isShow = true //$scope.isShow = false;
Ability manipulation by using ng-disable 

    <button ng-disabled="isDisable">Click me!</button>
Class manipulation by using ng-class 

    <div ng-class="textClass">Class manipulation</div> 

    In controller:
    $scope.textClass = "text-success";
Style manipulation by using ng-style 

    <div ng-style="textStyle">Style manipulation</div> 

    In controller:
    $scope.textStyle = {"color":"red"};
