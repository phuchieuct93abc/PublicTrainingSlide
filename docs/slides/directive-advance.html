Directive - Advance

How to pass value from outside into a directive

1. Defined scope option with param name 
 
     app.directive("searchField",function(){
     return {
            scope:{
                modelName:"="
            },
            templateUrl : "search-field.html",
            controller : function ($scope){
                 $scope.getValue = function(){
                    alert($scope.modelName)
                }
            }
        }  
    })
2.Pass value from outside into the directive 

    index.html
    <search-field model-name="parentModelName" ></search-field>
